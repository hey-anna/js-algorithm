## flow 요약

| 질문         | 정리                                                           |
| ------------ | -------------------------------------------------------------- |
| 배열은?      | `input + 1` 길이로 만들고, 인덱스 0\~input까지 사용            |
| 왜 3부터?    | `dp[1]`, `dp[2]`는 **초기값**이라 이미 있음 → `dp[3]`부터 계산 |
| 반복 범위는? | `i = 3`부터 `i <= input`까지 (input이 4면 3, 4 두 번 돈다)     |
