# 📌 만들 수 있는 경우 / 없는 경우 도식화

## 1. 만들 수 있는 경우 (출력 1)

### 예시

```
S = "B" ***
T = "ABBA"
```

### 역연산 흐름 (T → S)

```
"ABBA"   (끝이 A → A 제거)
   ↓
"ABB"    (끝이 B → B 제거 + 뒤집기)
   ↓
"BA"     (끝이 A → A 제거)
   ↓
"B"  === S와 같음 → 가능 → 출력 1
```

---

## 2. 만들 수 없는 경우 (출력 0)

### 예시

```
S = "AB" ***
T = "ABB"
```

### 역연산 흐름 (T → S)

```
"ABB"    (끝이 B → B 제거 + 뒤집기)
   ↓
"BA"     (끝이 A → A 제거)
   ↓
"B"  ≠ S → 불가능 → 출력 0
```

---

## 핵심 포인트

- **가능(1)**: T에서 역연산을 적용했을 때 S와 정확히 일치
- **불가능(0)**: 역연산을 적용하다 보면 S보다 짧아지거나, S와 다르게 끝남
